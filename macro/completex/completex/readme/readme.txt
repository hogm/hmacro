■インストール
　1.入手したアーカイブを秀丸のマクロフォルダに展開します。※
　2.[マクロ(M)]-[マクロ登録(E)...] で CompleteX.mac を登録します。
　3.[その他(O)]-[キー割り当て(K)...] で適当なキーに割り当てます。

　※マクロフォルダの場所は [その他(O)] - [動作環境(E)...] の [パス] - [環境] - 
　　[マクロファイル用のフォルダ] で確認できます。
　　無指定なら Hidemaru.exe と同じフォルダです。

■アンインストール
　1.インストール時と同じ方法でマクロ登録とキー割り当てを解除します。
　2.マクロフォルダの CompleteX.mac および CompleteX フォルダを削除します。

　レジストリは使ってません。

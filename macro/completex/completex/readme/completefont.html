CompleteFont.mac

フォント名補完マクロ　（CompleteX 用プラグイン）

ダウンロード

CompleteX に同梱されています。

概要

  • フォント名の補完機能を提供します。
  • システムにインストールされているフォント名が入力できます。

使い方

  • マクロフォルダ\CompleteX\CompleteFont.mac を実行します。
  • 半角英数字以外を含むフォント名を途中から補完するには、あらかじめフォント名の部分を範囲選択してから実行します。
  • 初回起動時にフォント辞書ファイルが自動的に作成されます。
  • フォント辞書ファイルは自動的に更新されません。フォントを追加／削除したときはフォント辞書ファイルを手動で削除してください。

設定

設定項目はありません。

お約束

動作環境、ライセンス、配布場所は CompleteX と同じです。
